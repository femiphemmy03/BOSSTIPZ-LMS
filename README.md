# Bosstipz LMS

Bosstipz LMS is a learning management system designed to help students maximize lecturers’ notes. It provides access to lecture videos, assignments, and TRCN practice sessions, all tailored to each student’s course and level.

## Features
- Watch lecturer videos linked to specific courses
- Attempt assignments for continuous learning
- Practice TRCN sessions for professional readiness
- Personalized experience based on course and level

## Getting Started
Clone the repository and follow the setup instructions to run locally.
